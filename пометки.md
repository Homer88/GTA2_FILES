# Пометки для разработки GTA 2

## Инициализация и выбор режима игры

### Текущая реализация
- Создан блок `InitWindow` с окном выбора режима (одиночная/многопользовательская)
- Выбор режима происходит через дополнительный параметр запуска (`-single`, `-multi`)
- В будущем планируется перенести выбор режима в главное меню как отдельный пункт

### Параметры командной строки
- `-single` / `-s` - одиночная игра
- `-multi` / `-m` - многопользовательская игра
- `-ip <address>` - IP сервера для мультиплеера
- `-port <number>` - порт сервера
- `-windowed` / `-w` - оконный режим
- `-fullscreen` / `-f` - полноэкранный режим
- `-res <width> <height>` - разрешение экрана
- `-nosound` - отключить звук
- `-nomusic` - отключить музыку

## DirectX 6.1
- Использовать только API DirectX 6.1
- Не использовать структуры DirectX 9 из дампов IDA
- Ориентироваться на данные из gta2.exe.map и gta2.exe.h

## Правила именования
- Использовать оригинальные имена функций из map-файла
- Для функций без имени использовать формат: `Func_0xXXXXXXXX`
- Структуры именовать по назначению: `Player`, `Ped`, `Car`, `Weapon`, `Gang`, `GameObject`, `Game`
- Все структуры должны быть упакованы через `#pragma pack(push, 1)`

## Адреса функций
Все функции должны содержать комментарии с точными hex-адресами из map-файла в формате:
```cpp
// Адрес функции из map-файла: 0x00401000
```
