# ⚠️ КРИТИЧЕСКИЕ ЗАМЕЧАНИЯ ПО ВЕРСИЯМ API

## 🎮 DirectX Версия
- **Целевая версия:** DirectX 6.1 (1998 год)
- **Статус:** Игра разрабатывалась под DirectX 6.1
- **Проблема:** В дампе IDA присутствуют структуры DirectX 9 (например, `IDirect3DDevice9Vtbl`, `IDirect3DTexture9`)
- **Действие:** 
  - Игнорировать структуры DirectX 9 при восстановлении кода
  - Использовать только определения, совместимые с DirectX 6.1
  - Проверять все графические структуры на соответствие эпохе выхода игры (1999 г.)

## 📂 Структуры
- При анализе структур типа `S...` проверять, не являются ли они переопределенными структурами DX9
- Ориентироваться на реальные размеры структур из ассемблерного кода, а не на имена из баз сигнатур IDA

## 📝 Для будущих чатов
- Всегда помнить про ограничение DirectX 6.1
- Не использовать современные аналоги структур без проверки совместимости
