# GTA2 Decompiled Code Project

## Описание
Этот проект содержит декомпилированный код игры GTA2, разбитый по модулям для упрощения работы.

## Структура проекта

```
gta2_project/
├── include/           # Заголовочные файлы
│   ├── types.h              # Базовые типы данных
│   ├── enums.h              # Перечисления
│   ├── forward_declarations.h # Предварительные объявления
│   ├── game.h               # Структуры и функции Game
│   ├── player.h             # Структуры и функции Player
│   ├── car.h                # Структуры и функции Car
│   ├── ped.h                # Структуры и функции Ped
│   └── common.h             # Главный заголовочный файл
│
└── src/             # Исходный код по модулям
    ├── game/        # Код связанный с Game
    ├── player/      # Код связанный с Player
    ├── car/         # Код связанный с Car
    ├── ped/         # Код связанный с Ped
    ├── mission/     # Код миссий
    ├── replay/      # Код повторов
    ├── registry/    # Код реестра
    ├── filemgr/     # Код файлового менеджера
    ├── sound/       # Звуковой код
    ├── render/      # Код рендеринга
    ├── object/      # Код объектов
    ├── gang/        # Код банд
    ├── sprite/      # Код спрайтов
    ├── weapon/      # Код оружия
    ├── ui/          # Код интерфейса
    └── main/        # Основной код
```

## Использование

Для использования в вашем проекте:

```c
#include "include/common.h"
```

или отдельные модули:

```c
#include "include/game.h"
#include "include/player.h"
```

## Примечания

- Код был декомпилирован с помощью Ghidra
- Некоторые имена полей и функций могут быть неточными
- Структуры могут требовать доработки
- Адреса функций сохранены в именах (например, FUN_003f1004)
