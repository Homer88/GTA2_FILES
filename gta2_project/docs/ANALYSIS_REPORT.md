## 3. Ключевые компоненты

### 3.1 Пешеходы (Ped)
- **Файл:** `src/ped/ped.c`
- **Функций:** 243
- **Ключевые функции:**
  - `Ped::SetSearchType()` - установка типа поиска цели
  - `Ped::GetCurrentState()` - получение текущего состояния
  - `Ped::SetHealthPlayer()` - установка здоровья
  - `Ped::SetOccupation()` / `GetOccupation()` - профессия
  - `S169::AreAnyPedActive()` - проверка активности пешеходов

### 3.2 Транспорт (Car)
- **Файл:** `src/car/car.c`
- **Функций:** 350+
- **Ключевые функции:**
  - `GetCarFlags()` / `GetCarDamageFlags()` - флаги состояния
  - `ResetCarPhysics()` - сброс физики
  - `BoardCarAsDriver()` - посадка водителя
  - `ApplyCarForce()` / `BrakeCar()` - управление

### 3.3 Структуры данных
- **S1 (LinkedList):** Базовый связанный список (~90 байт)
- **S2 (Vector3D):** Вектор для координат
- **S3 (Matrix3D):** Матрица трансформации
- **S17 (PedStats):** Статистика пешехода (~12 КБ)
- **S160 (TrafficLight):** Светофор (~6 КБ)
- **S169 (SpawnPoint):** Точка появления (~2.1 КБ)

## 4. Проблемные зоны

1. **Непереименованные функции:** ~1300 функций всё ещё имеют имена вида `FUN_xxxxxx`
2. **Глобальные переменные:** Много глобального состояния, затрудняющего тестирование
3. **Жёсткая привязка:** Прямая зависимость от адресов памяти
4. **Отсутствие RTTI:** Нет информации о типах во время выполнения
