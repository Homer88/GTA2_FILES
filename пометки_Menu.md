## Планы на будущее

### Параметр запуска для выбора режима
В будущем планируется добавить параметр командной строки для выбора режима игры:
- `-single` - одиночная игра
- `-multi` - многопользовательская игра

Это позволит избежать создания отдельных блоков кода для каждого режима, а вместо этого использовать единый блок меню с дополнительным пунктом выбора режима.

**Файл для обновления:** `InitWindow.h` / `InitWindow.cpp`
**Статус:** Требуется реализация в будущем
